Connect-4 by kvannotten

Connect 4 in C, fully working version, need SDL, some memory leaks still exist

Connect 4 school project. C-project. Assignment was to make a console game. We thought console games were lame and did something more. This is the result, a memory leaking dragon of a game, which is enjoyable nonetheless. Extra features are present, computer cheats, high score list, ... Enjoy!

Dependencies

SDL, SDL-mixer, SDL-font, SDL-image

Install

Install SDL, SDL-mixer, SDL-font, SDL-image. Compile using the makefile. Play.
